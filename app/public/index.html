Register domain on DLS to be able to add/remove sellers

Make sure to have the DNS TXT record: dlspubkey, with your ethereum public key as the value.

Example
      dlspubkey.example.com = "0x123...abc"

Only the private key pair of the public key will be able to sign transactions to update list of sellers for the domain.

Once you've done that, enter your domain and submit.
